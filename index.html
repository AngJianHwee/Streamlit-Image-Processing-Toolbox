Image Processing Toolbox

Welcome to the Image Processing Toolbox

As a deep learning researcher, I often found myself needing to apply several filters and transformations to images during the data preprocessing stage. To streamline this and make the process more intuitive, I created this toolbox using Streamlit. This site enables the quick adjustment and visualization of different image processing techniques, improving the efficiency and understanding of the effects each transformation can have.

Base64 Encoding

Encode an image to base64 format and decode it back to its original form. Features:

  • Upload an image to encode
  • Display the encoded string
  • Decode the string to view the original image
Try it out!

Gaussian Blur

Interact and learn about the impact of varying parameters on the Gaussian blur operation. Features:

  • Control kernel size through a slider
  • Standard deviation alteration via another slider
  • Interactive display of effect on image
Try it out!

Dilation

Apply dilation to an image and observe the effects of varying the kernel size. Features:

  • Interactive display of the dilated image
